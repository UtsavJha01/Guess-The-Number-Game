A simple CLI-based Guess The Number Game built using Python.

------------------------------------------------------------

PROJECT DESCRIPTION

This project is a beginner-friendly Python game that demonstrates:

- Random number generation
- While loops
- Conditional statements
- User input handling
- Attempt tracking

The game continues until the correct number is guessed.
At the end, it displays the total number of attempts taken.

------------------------------------------------------------

TECHNOLOGIES USED

- Language: Python 3
- Module: random
- Interface: Command Line (CLI)
